Instructions

  • First, please click on App on the navigation bar to go to functionality.

  • On the first tab, you'll see the input types of text and file and also two buttons: Submit and Reset.

    A user can use both: an input types of text and file to upload the json schema. If the user uploads a file, input will be cleared (if there is text) and disabled.

    The submit button is disabled at the beginning, and if only the user enters or uploads a JSON schema, it becomes active. After submitting, it switches to the second tab, and if the user uploaded the JSON schema in the correct format, a dynamic form will be created.

    Clicking on the reset button will clear and disable the input field, the file uploader, and the form created on the second tab.

  • On the second tab, the user can see the created reactive form. Elements are: input, checkbox, radio, textarea, and the plain text as it was asked in the assignment.

    Additionally, users can add validations to the form fields. I also added convenient error messages for the validations.

    • You can find the JSON file in the email attachments and also in the project in the src/assets/json folder.

    If you need me to add anything else, or if any errors occur, please let me know! :)
